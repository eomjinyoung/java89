주요 이벤트 - key 이벤트

- 키보드의 키를 누르거나 뗐을 때 발생하는 이벤트
- input 이벤트
  => 키보드로 값을 바꿀 때 마다 호출되는 이벤트
- keydown 이벤트
  => 키보드에서 키를 누르면 발생한다. 
  => 계속 누르고 있으면 연속해서 발생한다.
  => 입력 상자의 값이 변경되기 전에 발생한다. 
     따라서 입력 상자의 값을 꺼내면 현재 키 값이 적용되지 않은 상태의 값이다.
- keyup 이벤트
  => 눌린 키를 놓을 때 발생한다.
- keypress 이벤트
  => 키를 누르면 한 번 발생한다.
  => 계속 키를 누르고 있어도 한 번만 발생한다.
