자바스크립트 내장(built-in) 객체

- 자바스크립트에서 기본을 제공하는 주요 객체들
- 기본 객체
  Object    : 자바스크립트 기본 객체를 생성하거나 다룰 때 사용한다.
  Number    : 숫자 다룰 때 사용한다.
  Math      : 수학 함수를 포함하고 있다.
  Date      : 날짜를 다룰 때 사용한다.
  String    : 문자열을 다룰 때 사용한다.
  RegExp    : 문자열에서 특정 패턴을 검사할 때 사용하는 정규 표현식을 다룬다.
  Array     : 배열을 다룰 때 사용한다.
  JSON      : JSON 형식의 문자열을 다룰 때 사용하낟.
