DOM API 사용 - CSS 셀렉터를 이용하여 태그 찾기

- 태그를 정교하게 찾고 싶다면, CSS 셀렉터 명령을 사용하라!
- document.querySelector(CSS 셀렉터 명령)
  => 리턴 값은 한 개의 엘리먼트이다.
  => 만약 여러 개를 찾았다면 그 중에 0번째 태그만 리턴한다.
  
- document.querySelectorAll(CSS 셀렉터 명령)
  => 리턴 값은 HTMLCollection이다.
  => 여러 개를 찾을 때 사용하라!
