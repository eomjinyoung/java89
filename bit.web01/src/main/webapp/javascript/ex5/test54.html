DOM API 사용 - 속성 추가하기

- 개발자가 HTML을 제어할 용도로 새 속성을 추가할 때는
  다음과 같은 규칙으로 속성명을 작성하라! (규칙이 아니라 권고사항이다.)
  => data-속성명
- 해당 태그에 이미 존재하는 속성이라면 다음과 같이 속성을 추가할 수 있다.
  => 태그객체.속성명 = 값;
  => 태그객체["속성명"] = 값;
- 해당 태그에 존재하지 않는 속성을 추가하려면 위와 같이 할 수 없다.
  다음과 같이 함수를 호출해야 한다.
  => 태그객체.setAttribute("새속성명", "값");
