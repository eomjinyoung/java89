DOM API 사용 - ID로 태그 찾기

- DOM(Document Object Model) API는 웹브라우저에서 생성한 태그를 다룰 때 사용한다.
- 웹브라우저는 HTML을 읽을 때 다음의 절차에 따라 화면을 준비한다.
  1) HTML을 해석한다.
  2) 웹브라우저 메모리에 DOM Tree 구축
     - 각 태그를 자바스크립트 객체로 만들어 트리 구조로 구성한 것. 
  3) 렌더 트리 구축
     - DOM Tree 중에서 당장 화면에 출력할 태그만 추려 놓은 트리.
  4) 렌더 트리를 화면에 출력
