# 깃 사용법
1) 저장소 복제
> git clone [저장소 URL]

2) 로컬에 저장할 파일 목록 만들기
> git add .
> git add [파일명]

3) 파일 목록에 등록된 파일을 로컬 저장소에 백업하기
> git commit -m '백업메시지'

4) 브랜치 목록 조회
> git branch

5) 브랜치 교체
> git checkout [브랜치명]

6) 특정 태그를 브랜치로 만들기
> git checkout -b [새로만들브랜치명] [태그명]

7) 브랜치 삭제
> git branch -d [브랜치명]

8) 서버 저장소에서 로컬 저장소로 변경한 것 가져오기
> git pull
